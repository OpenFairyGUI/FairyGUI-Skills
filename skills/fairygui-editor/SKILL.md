# FairyGUI Editor Skill

## Workflow

1. Confirm the target engine (Unity, UE, Cocos, or custom) and FairyGUI version.
2. Identify the editor task: package setup, component layout, transitions/animations, resource import/export, or publishing.
3. If the task is detailed or tool-specific, read the relevant reference file:
   - Editor workflows: references/editor-workflows.md
   - Publishing/build: references/publish-build.md
   - Troubleshooting: references/troubleshooting.md
4. Provide step-by-step guidance and list required editor actions or settings.

## Output expectations

- Provide explicit menu paths, panel names, and key settings when possible.
- Call out version-specific differences if known.
- Ask for screenshots or error messages if the issue is unclear.
